# Explainers for glossary terms

Working list: terms from nuggets and suggested explainer links. Add URLs as you find them; when we support `#explainer` in nuggets, copy the link into the nugget where the term is introduced.

**Goal:** Short + best. Prefer under 10 min, ideally ~5. Always note duration with each link so we can show it in the UI (e.g. "Explain (13 min)"). When choosing: length, ratings (if you check), and source reputation (Numberphile, Veritasium, PBS Space Time, etc.). Source is for curation only — not shown to the user.

Format: **Term** — nugget number(s) — then one line per video: `- https://... (X min, Title)` or `- https://... (Title)`. Use duration when known; omit "check duration" — use title/source only so the build does not need to strip it.

**Summary:** Added roughly 50 explainer video links, balanced across term groups: 001/027 (paradigm, Kuhn, hard problem, caloric, normal science, presentism, Whig history, animism, vitalism, quantum field; ~18 links), 008 (EM field, radiation, diffraction, prehension; ~8 links), 006 (process/Whitehead; ~2 links—output, prediction, “just” have no good short standalone explainers), 021/022 (category mistake, intuition pump, locked-in, vegetative state, presume competence, FC; ~14 links), and 020 (Gödel, Orch-OR, computationalism; ~8 links). Prefer under-10-min clips; source noted in parentheses after duration for curation only. Where duration was unverified, "(check duration)" or "(duration unknown)" is used.
