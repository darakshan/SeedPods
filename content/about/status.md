# SeedPod Status Values

Each seedpod carries a status that reflects how complete and polished it is. From most to least complete:

| Status | Meaning |
|--------|---------|
| `draft` | Complete; word count checked. Any status beginning with "draft" qualifies. |
| `rough` | All four sections present but need a lot of work. |
| `partial` | Two or three sections have content. |
| `prelim` | One section, complete. |
| `proto` | One section, rough. |
| `empty` | No sections yet. |
